# Scientific Research

A repository containing implementations, models, and analyses of scientific articles authored and co-authored by **André Gambogi**.


## 📚 Research Publications

This repository includes code, models, and comprehensive resources related to the following scientific research:

### 1. 🧬 [A Machine Learning-Guided Approach for a Multi-Epitope HIV Vaccine Design](./A%20Machine%20Learning-Guided%20Approach%20for%20a%20Multi-Epitope%20HIV%20Vaccine%20Design/)

An innovative computational approach leveraging machine learning for rational design of multi-epitope HIV vaccines. This work combines advanced bioinformatics techniques with artificial intelligence algorithms to optimize epitope selection and vaccine efficacy prediction.

- **Research Focus:** Computational vaccine design, epitope prediction, immunoinformatics
- **Technologies:** Machine Learning, Bioinformatics, Sequence Analysis, Optimization Algorithms
- **Key Contributions:** Enhanced epitope selection framework, ML-based immunogenicity prediction
- **Relevant Fields:** Immunology, Virology, Computational Biology

### 2. 🧪 [DEP: Dual-Path Embeddings for Protein Toxicity Classification](./DEP/)

A sophisticated deep learning model featuring dual-pathway embeddings for protein toxicity classification. DEP combines local and global attention mechanisms to achieve superior performance in toxicity prediction tasks.

- **Research Focus:** Protein analysis, toxicity prediction, deep learning architectures
- **Technologies:** Deep Learning, PyTorch, Transformer Networks, Neural Embeddings, Attention Mechanisms
- **Key Contributions:** Dual-path neural architecture, efficient attention mechanisms, robust classification model
- **Relevant Fields:** Computational Toxicology, Biochemistry, Machine Learning
